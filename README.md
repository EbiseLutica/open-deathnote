# [オープンソースデスノート](https://xeltica.github.io/open-deathnote/)

GitHubで管理するオープンソースのデスノートです。

プルリクエストを発行しないと名前を書き込めないという特徴以外は、おそらく本来のデスノートと同じです。

ちなみに作者はデスノートを読んだことも見たこともありません。
